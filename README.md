# SyncFolder

 <img src=https://github.com/DFortmann/SyncFolder/raw/master/SyncFolder%20interface.png alt="SyncFolder" width=70% height=70%>
 
 SyncFolder is a backup utility for Windows. <br>
 You can copy, overwrite or synchronize folders, while only new or changed files will be processed.
 
 You can download a compiled version here: https://github.com/DFortmann/SyncFolder/raw/master/SyncFolder.zip
